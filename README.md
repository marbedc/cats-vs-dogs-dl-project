# Cats vs Dogs Deep Learning Project

## Overview

This project implements image classification models to distinguish between cats and dogs using deep learning.

## Models

* Custom CNN (baseline)
* Pretrained model (ResNet-18)

## Dataset

Kaggle Dogs vs Cats dataset

## Team Roles

* Person 1: Custom CNN
* Person 2: Pretrained model
* Person 3: Evaluation and analysis

## How to Run

1. Install dependencies
2. Download dataset (not included in repo)
3. Run training scripts

## Notes

* Dataset and model weights are not included due to size
